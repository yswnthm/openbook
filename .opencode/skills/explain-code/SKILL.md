# Explain Code

When asked to explain code, analyze the relevant files and provide a clear, concise explanation of:

1. What the code does
2. Key functions and their purposes
3. How different parts interact
4. Any notable patterns or techniques used

Focus on the most important aspects first. Use the code itself as reference.
